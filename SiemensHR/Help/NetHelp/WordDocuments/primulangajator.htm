Primul angajator

După instalarea aplicaţiei, dacă există un angajator în baza de date, utilizatorul este redirectat către prima pagină a aplicaţiei şi toate acţiunile sunt făcute pentru acest angajator.

Dacă nu există un angajator în baza de date, utilizatorul este redirectat către pagina pentru introducerea primului angajator.

Toate acţiunile sunt realizate folosind angajatorul nou adăugat.

Datele unui angajator sunt grupate în cinci categorii :

o Date angajator

o Conturi angajator

o Adresa angajator

o Domenii de activitate

o Punct de lucru

o Reprezentanti legali

Tab-ul Date angajator conţine datele despre angajatorul curent. Pe acest formular este obligatorie specificarea numelui angajatorului, CUI-ului, numărului de înregistrare ORC şi a numărului de telefon.

Tab-ul Conturi angajator conţine date despre conturile angajatorului curent. Pe acest formular este obligatorie specificarea unui cont pentru angajator. Conturile sunt vizualizate în lista de pe formular.

Pentru a adăuga un cont pentru angajator trebuie specificate banca de care aparţine contul,  numărul contului în format vechi, numărul cont IBAN, tipul monezii şi specificaţia dacă este cont activ sau nu. După ce au fost specificate aceste date despre un cont, pentru adăugarea lui în listă se apasă butonul “Adaugă cont”.

Pentru a modifica un cont, se selectează din lista cu conturi cel ce se modifică. Controalele se completează cu datele corespunzătoare acestuia. Pentru a salva modificările făcute se apasă butonul “Modifică date cont”.

Pentru adăugare şi modificare, pentru aceeaşi monedă nu pot exista două conturi active şi nu pot exista două conturi identice.

Pentru a şterge un cont, se selectează din lista cu conturi cel ce se şterge şi se apasă butonul “Şterge cont”. Nu se poate şterge un cont care este activ.

Tab-ul Adresa angajator conţine adresa angajatorului curent. Pe acest formular este obligatorie specificarea localităţii, străzii, numărului şi codului poştal de care aparţine angajatorul. Pentru codul poştal este necesară specificarea unei valori numerice.

Tab-ul Domenii de activitate conţine domeniile de activitate ale angajatorului curent. Pe acest formular este obligatorie specificarea unui domeniu de activitate pentru angajator. Pentru a asocia domeniile de activitate unui angajator, se aleg domenii din listă şi se adaugă în lista celor asociate angajatorului. Pentru a realiza aceste operaţii se folosesc butoanele cu săgeţile corespunzătoare. Ştergerea domeniilor se realizează tot cu ajutorul acestor butoane.

Tab-ul Punct de lucru conţine punctele de lucru ale angajatorului curent. Pe acest formular este obligatorie specificarea unui punct de lucru pentru angajator.

Pentru a adăuga un punct de lucru unui angajator, se introduce numele acestuia, se selecteaza casa de asigurari implicita şi se apasă butonul “Adaugă”. Nu se pot adăuga două puncte de lucru identice.

Pentru a şterge un punct de lucru, se selectează cel ce se şterge şi se apasă butonul “Şterge”. Pentru a şterge toate punctele de lucru se apasă butonul “Şterge tot”. Nu se pot şterge puncte de lucru care sunt asociate unor angajaţi.

Tab-ul Reprezentanti legali conţine reprezentantii legali ai angajatorului curent. Pe acest formular este obligatorie specificarea pentru fiecare functie a unui reprezentant legal.

Pentru a adăuga un reprezentant legal unui angajator, se introduce numele si prenumele acestuia, se selecteaza functia şi se apasă butonul “Adaugă”.

Pentru a modifica un reprezentant legal, se selectează cel dorit şi se apasă butonul “Şterge, dupa care se adauga din nou”. Pentru a şterge toti reprezentantii legali se apasă butonul “Şterge tot”.

După introducerea modificărilor salvarea datelor se face apăsând butonul “Salvează angajator”. Pentru a naviga între tab-urile cu date despre angajator se pot apăsa butoanele “Back” şi “Next” sau se poate da click pe tab-ul dorit.

Utilizatorul este apoi redirectat către prima pagină a aplicaţiei. Pentru actualizarea datelor angajatorului se alege din meniu opţiunea Administrare -> Angajator -> Angajatori.
