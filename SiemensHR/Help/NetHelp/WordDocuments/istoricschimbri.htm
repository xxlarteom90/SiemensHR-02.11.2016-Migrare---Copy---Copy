Istoric schimbări

Această operaţie este accesibilă din interfaţă dacă se urmăresc paşii : Login à Angajaţi à Editare angajat à Istorice angajat à Istoric schimbări.

Pentru a adăuga o înregistrare nouă se dă click pe tab-ul “Adaugă schimbare”. Se va deschide o fereastră care va permite introducerea unui nou set de date. Se pot schimba datele angajatului numai dacă data încheierii contractului este ulterioară datei începerii schimbării.

Pentru a şterge o înregistrare, se dă click pe înregistrarea dorită şi aceasta este ştearsă. Ştergerea unei înregistrări se face numai după confirmarea utilizatorului. Nu se poate şterge o înregistrare precedentă lunii active, la fel cum nu se poate şterge o înregistrare dacă este singura din lista cu schimbări.

Dacă se doreşte anularea se apasă butonul “Anulează”.
