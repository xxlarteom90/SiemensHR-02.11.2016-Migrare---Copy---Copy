Adăugare angajat

Această modul  al aplicaţiei permite adăugarea unui angajat.

Adăugarea angajaţilor se face după instalarea aplicaţiei şi de fiecare dată când se fac noi angajări.

Operaţia este accesibilă din interfaţă dacă se urmăresc paşii : Login à Angajaţi à Apăsarea butonului “Angajat nou “.

Se deschide o pagină care conţine formularul pentru adăugarea unui angajat nou.

După introducerea tuturor datelor la apăsarea butonului “Salvează date” este adăugat un nou angajat. Dacă se doreşte o anulare se apasă butonul “Înapoi”.  Pentru înaintarea la următorul tab se apasă butonul “Next” sau se dă click pe tab-ul dorit. Pentru revenirea la un tab anterior se apasă butonul “Back”.

Primul tab din formularul pentru adăugarea unui angajat este cel de Identificare.

Pe acest formular este obligatorie introducerea numelui şi prenumelui angajatului.  Pentru câmpul care specifică numărul de copii trebuie introdusă o valoare numerică. Dacă pentru angajatul care se adaugă, nu există o invaliditate corespunzătoare în lista cu invalidităţi, atunci se poate adăuga una nouă prin apăsarea butonului din partea dreaptă a listei cu invalidităţi. Se procedează analog pentru titlu şi studii. Anul absolvirii are de asemenea o restricţie prin care se pot introduce valori numai din intervalul 1900 – 2500. Pentru numărul de telefon este obligatorie introducerea unei valori numerice, iar pentru adăugarea unei poze se apasă butonul “Browse” şi apoi se selectează poza dorită.

Al doilea tab din formularul pentru adăugarea unui angajat este cel de Domiciliu – Reşedinţă.

Dacă pentru angajatul care se adaugă nu există o ţară corespunzătoare în listă, se poate adăuga una nouă prin apăsarea butonului din partea dreaptă a listei care conţine ţări. Analog se procedează şi pentru adăugarea unui nou sector sau judeţ. Pe acest formular este obligatorie introducerea numelui localităţii, a străzii, a numărului şi al codului poştal atât pentru domiciliu cât şi pentru reşedinţă.

Următorul tab pentru adăugarea unui angajat este cel pentru Date de identificare.

Dacă angajatul este de naţionalitate română, va apărea pe formă şi un formular pentru completarea datelor despre buletinul de identitate. Pentru completarea datei de naştere, a datei de eliberare a buletinului, a paşaportului sau a legitimaţiei de şedere se deschide un calendar din care se alege data dorită. Pe acest formular este obligatorie introducerea datei de naştere, a localităţii, judeţului şi ţării de naştere,  prenumelui mamei şi al tatălui,  CNP-ului, datelor despre buletinul de identitate şi a datelor despre paşaport. La introducerea CNP-ului se face validarea acestuia pentru a nu se introduce date greşite.

Şi aici se poate adăuga o nouă ţară sau sector dacă nu există pentru angajatul curent.

Următorul tab pentru adăugarea unui angajat este cel pentru Date angajare.

Pe acest formular se poate adăuga o nouă categorie de încadrare, prin apăsarea butonului din dreapta listei cu categorii, în cazul în care nu există una specifică angajatului, se poate adăuga o nouă funcţie sau un nou departament.

Câmpurile obligatorii pe acest formular sunt: marca, programul de lucru, perioada angajării, salariul de încadrare, numărul de zile de concediu de odihnă pe an şi numărul de zile de concediu suplimentar pe an.

Datele introduse pe acest formular sunt supuse unui set de restricţii şi anume: marca trebuie să fie unică, deci nu poate fi aceeaşi la mai mulţi angajaţi, pentru programul de lucru trebuie introdusă o valoare numerică,  numărul de telefon şi adresa de email trebuie să fie valide, sumele care se introduc trebuie să aibă valori numerice.

Pentru Perioada angajării se selectează intervalul de timp corespunzător tipului de perioadă ales. Pentru selectarea datei de început sau de sfârşit se dă click pe căsuţa pentru dată şi se deschide un caledar din care se va alege data. Analog pentru data de la Date majorare.

În cazul în care nu este primul loc de muncă al angajatului pe formă vor apărea controale pentru introducerea venitului acestuia pe ultimele şase luni. Aceste valori vor fi necesare pentru calcularea mediei pentru concediul de boală.

    

Următorul tab pentru adăugarea unui angajat este cel pentru Date PSI.

Pe acest formular se poate adăuga diferite drepturi şi obligaţii ale părţilor privind sănătatea şi securitatea în muncă.

Dacă se alege opţiunea “Da” din liste, vor apărea controale pentru adăugarea de informaţii suplimentare referitor la dreptul sau obligaţia pentru care s-a ales opţiunea “Da”.

Ultimul tab pentru adăugarea unui angajat este cel pentru Alerte speciale.

După ce au fost completate datele pentru o alertă, adăugarea ei se face prin apăsarea butonului “Adaugă”. Ştergerea unei alerte se face astfel: se selectează din listă alerta care se şterge şi se apasă butonul “Şterge alertă”. Datele obligatorii pentru o alertă sunt descrierea, data expirării şi numărul de zile până la declanşarea alertei.

În cazul apariţiei de erori la completarea datelor pentru un angajat, utilizatorul aplicaţiei va primi mesaje de avertizare care specifică tipul erorii întâlnite.
