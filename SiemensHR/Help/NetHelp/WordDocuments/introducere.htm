Introducere

Produsul HR Toolkit este destinat departamentelor de tipul Business Administration ce doresc să-şi creeze o platformă de lucru destinată administrării datelor personale ale angajaţilor, precum şi a celor legate de pontaj şi de calculul salarial. 

Scopul produsului este de a oferi o soluţie software care să funcţioneze ca platformă de lucru destinată departamentelor de tipul Business Administration. Această platformă trebuie să integreze funcţionalităţi menite să optimizeze desfăşurarea unor activităţi precum: gestiunea datelor despre angajaţi, a pontajului, calculul salarial, generarea rapoartelor specifice.

HR Toolkit este o aplicaţie de tipul Enterprise Resource Planning realizată folosind tehnologiile Microsoft. Arhitectura aplicaţiei este de tipul client – server. Produsul oferă integrarea cu Windows authentication si Form authentication si  şi pentru transferul de date este utilizat protocolul Secure Socket Layer, care efectuează criptarea datelor, oferind astfel un înalt nivel de securitate.

Fig. 1 Arhitectura aplicaţiei HR Toolkit

HR Toolokit presupune existenţa unei infrastructuri care să suporte rularea aplicaţiilor web: reţea funcţională, disponibilitatea protocolului TCP/IP, precum şi a  unui calculator disponibil să ruleze serverul de baze de date, Microsoft SQL Server, şi aplicaţia web. Nivelul de persistenţă al datelor este asigurat de o bază de date SQL Server şi de un set de componente software ce trebuie instalate pentru a putea folosi aplicaţia. Aplicaţia presupune, ca şi infrastructură software necesară, existenţa sistemului de operare Windows 2000/XP pe calculatorul care rulează aplicaţia şi baza de date existente.

În cadrul acestui manual vor fi prezentate următoarele: descrierea detaliată a funcţionalităţilor, instrucţiunile de instalare şi dezinstalare, precum şi cerinţele de sistem necesare unei rulări în parametrii optimi.
