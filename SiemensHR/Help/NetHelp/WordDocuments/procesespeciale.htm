Procese speciale

Procedura de backup este următoarea:

-  zilnic se va face un backup la baza de date;

-  la sfârşitul săptămânii se vor inscripţiona ultimele backup-uri ale bazei de date pe cd / dvd;

-  se va face backup la proiectul existent în SourceSafe.

Toate fişierele de backup se vor arhiva pe support magnetic (TAPE sau CD / DVD) de către administratorul serverului aplicaţiei.
