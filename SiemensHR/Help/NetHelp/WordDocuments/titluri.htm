Titluri

Această operaţie este accesibilă din interfaţă dacă se urmăresc paşii : Login à Administrare à Angajaţi à Studii.

Se pot face următoarele operaţii:

•    Adăugare

•    Modificare

•    Ştergere

 

Adăugarea

Dacă se dă click pe butonul “Titlu angajat nou” se va deschide o fereastră care va permite introducerea unui nou set de date.

Datele introduse vor fi salvate după apăsarea butonului “Salvează titlu”.

Pentru anulare se poate selecta butonul „Înapoi“.

 

Modificarea / Ştergerea

Editarea unui titlu se face dând click pe setul de date care se doreşte a fi modificat.

Atât pentru adăugare, cât şi pentru modificare, specificarea denumirii şi simbolului titlului este obligatorie.

Se permite atât modificarea unui titlu, cât şi ştergerea acestuia. Nu se poate şterge un titlu dacă există înregistrări ataşate lui sau dacă acesta este ultimul din listă.

După introducerea modificărilor salvarea datelor se face apăsând butonul “Modifică titlu”, iar ştergerea se face prin apăsarea butonului “Şterge titlu”. Dacă se doreşte anularea se apasă butonul “Înapoi”.
